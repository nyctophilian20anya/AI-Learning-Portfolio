\# Week 2 — Using Multiple LLMs \& Comparing Outputs



\## Objective

Run the same prompt on ChatGPT 4.0 and Claude (Microsoft Copilot),

record results, and compare and analyse the outputs.



\## Exercise Steps Followed

1\. Accessed ChatGPT 4.0

2\. Ran the prompt

3\. Recorded results

4\. Added Claude (Microsoft Copilot)

5\. Ran the same prompt

6\. Recorded results

7\. Compared and analysed both outputs



\## Prompt Used

Convert a Generic Telecom HLSO (High-Level Solution Overview) 

into a detailed HLS (High-Level Solution) for 

Customer Order Provisioning.



\## Tools Used

| Tool | Model |

|------|-------|

| ChatGPT | GPT-4.0 |

| Microsoft Copilot | Claude |



\## Files

| File | Description |

|------|-------------|

| HLSO\_Generic\_Telecom.pptx | Source HLSO document used as input |

| HLS\_Output\_ChatGPT.docx | HLS generated by ChatGPT |

| HLS\_Output\_Claude.docx | HLS generated by Claude (Copilot) |

| LLM\_Comparison\_Analysis.docx | Final comparison and analysis |



\## Key Takeaway

Claude produced a more enterprise-aligned HLS with better 

error handling categorisation and measurable success criteria.

ChatGPT produced cleaner prose but lacked depth in critical sections.
